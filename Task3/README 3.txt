Project Name: Student Management System

Language: C Programming

Description:
A simple Student Management System developed in C to store and display student records using file handling and structures.

Features:

* Add Student Records
* Display Student Records
* Store Roll Number, Name, and Marks
* File Handling for Data Storage
* Menu-Driven Interface

Sample Input:
Enter Roll No: 101
Enter Name: Prachiti
Enter Marks: 95

Sample Output:
Student added successfully.

Student Records:
Roll No: 101
Name: Prachiti
Marks: 95.00
